# Hangman
A simple hangman game that is played in a terminal.

## Installation
### Arch linux
Download the PKGBUILD file, create a new directory, move the PKGBUILD file into that directory and cd into it. Then:
``` 
makepkg -si PKGBUILD
```

## Uninstall
### Arch linux
```
sudo pacman -R hangman
```
